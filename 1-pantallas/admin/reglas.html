Reglas de casos

Cuando una conversación cumpla una regla, la IA abrirá un caso para que un agente lo atienda.

Te incluimos 5 reglas habituales para que empieces. Edítalas, desactívalas o crea las que necesites para tu negocio.

Cliente con reclamo

Plantilla

El cliente reclama, se queja, expresa insatisfacción con un producto o servicio, o reporta un problema.

Crea un caso: Reclamo

Cliente quiere hablar con un humano

Plantilla

El cliente pide explícitamente hablar con una persona, con un encargado, o dice que no quiere seguir hablando con un bot.

Crea un caso: Derivación solicitada

Pedido grande o al por mayor

Plantilla

El cliente consulta por pedidos grandes, al por mayor, para eventos, o solicita presupuesto personalizado.

Crea un caso: Venta consultiva

Tema sensible o fuera de competencia

Plantilla

El cliente pregunta sobre temas legales, médicos, fiscales, denuncias, o cualquier asunto delicado fuera de lo que el negocio puede responder.

Crea un caso: Tema sensible

Cliente molesto o agresivo

Plantilla

El tono del cliente es claramente molesto, agresivo, ofensivo, o muestra señales de mucha frustración.

Crea un caso: Atención urgente

Nueva regla

La IA entiende intenciones, no necesitas escribir palabras exactas.

Esta etiqueta aparecerá en la bandeja de casos.
